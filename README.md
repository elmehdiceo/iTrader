**Software to be installed :**
- NodeJS
- Gradle
- JDK8
- Android SDK
- Android Studio
- Google Chrome
- Global Npm Modules:
- cordova
- gulp cli
- boxer
- ionic
- ios-sim
- Genymotion (Android emulator replacement) `
- XCode
- XCode Command Line Tools
